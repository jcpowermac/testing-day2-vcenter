# Tech Stack

- Language: Go (1.22+)
- Test framework: Ginkgo v2 / Gomega
- Build/test: Makefile targets (see CLAUDE.md)
- Dependency management: Go modules (`go.mod`)
- Platform: Linux (Fedora)
- OpenShift client libraries: openshift/api, openshift/client-go
- Kubernetes client libraries: k8s.io/api, k8s.io/client-go, k8s.io/apimachinery
- Cloud config format: YAML parsed with gopkg.in/yaml.v3
